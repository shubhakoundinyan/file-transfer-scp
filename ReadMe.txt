OVERVIEW:

This is a script that helps to securely transmit files between 2 Linux machines.

PRE-REQs:


1.	OpenSSL Server: This needs to be installed prior to the execution of the script. 

You can install it using this command :-	sudo apt-get install openssl


2.	SSHPass: You can install it using this command :-	sudo apt-get install sshpass
